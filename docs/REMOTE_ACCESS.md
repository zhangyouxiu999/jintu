## 2. 确保系统不掉线 (核心设置)

因为您的 Mac 充当了服务器，**建议始终连接电源线**，并确保系统不进入深度休眠。

- **电源连接**：必须始终连接电源线，否则 macOS 会为了省电强制切断 Wi-Fi。
- **自动防休眠**：我已经为您配置了 `keep-awake` 守护任务。只要电脑开着，它就会阻止系统进入深度睡眠。
- **合盖设置**：虽然我已经配置了防休眠任务，但 macOS 系统的安全机制可能会在长时间合盖后降低网络优先级。如果发现外网访问变慢，请尝试稍微掀开一点盖子或保持电脑处于开启状态。

## 3. 后台运行管理 (PM2)

我已使用 **PM2** 将项目、穿透工具和防休眠指令都挂在了后台。

### 如何查看状态？

在终端输入 `pm2 status`，您应该能看到三个 `online` 状态的任务：

- `jintu-admin`: 考勤系统主程序
- `ngrok`: 外网穿透工具
- `keep-awake`: 强制保持系统唤醒（支持合盖运行）

### 常用命令：

- **查看实时日志**: `pm2 logs` (如果打不开网页，先看这里有没有报错)
- **重启所有服务**: `pm2 restart all`
- **停止所有服务**: `pm2 stop all`

---

## 4. 故障排查

如果你发现网页打不开：

1. **检查网络**：确保 Mac 连接了稳定的 Wi-Fi 或有线网。
2. **检查休眠**：看 Mac 是不是黑屏休眠了，点亮屏幕试试。
3. **重启服务**：打开终端，输入 `pm2 restart all`。
4. **验证本地访问**：在 Mac 浏览器打开 `http://localhost:3000`，如果本地能开但外网不行，说明是 `ngrok` 的问题。
