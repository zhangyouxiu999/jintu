# 考勤 App

基于 `Vite + React + TypeScript + Capacitor` 的轻量考勤应用，面向 iOS / Android 本地使用场景，数据默认保存在设备本地。

## 技术栈

- `Vite`
- `React 18`
- `TypeScript`
- `Tailwind CSS`
- `Capacitor 6`

## 本地运行

```bash
npm install
npm run dev
```

## 打包与同步

```bash
npm run cap:sync
```

```bash
npm run android
```

```bash
npm run ios
```
